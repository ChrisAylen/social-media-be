## Installation
    
'npm i' to install the necessary dependencies.

Create a file named .env with the following contents:

```
DB_NAME=yourdbname
DB_USER=yourdbuser
DB_PASSWORD=yourdbpassword
DB_HOST=yourdbhost
DB_PORT=3306
PORT=3001
```
